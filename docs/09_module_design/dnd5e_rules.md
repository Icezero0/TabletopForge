# DND5E 规则计算模块设计

版本：v0.2  
状态：Draft

---

# 1 模块目标

DND5E 规则计算模块用于提供轻量规则辅助，降低玩家手动计算成本。

本模块不在第一阶段实现完整规则库，只提供高频基础计算。

---

# 2 第一阶段能力

- 属性调整值计算
- 熟练加值计算
- 技能加值计算
- 豁免加值计算
- 先攻加值计算
- 法术 DC 计算
- 法术攻击加值计算
- 骰子表达式支持
- 优势 / 劣势计算
- 短休 / 长休资源恢复规则

---

# 3 规则集扩展

房间保留 ruleset 字段。

当前实现：

```text
DND5E
```

后续可扩展：

```text
CUSTOM
COC7E
PF2E
```

---

# 4 后端组织

建议组织为：

```text
rulesets/
├── base/
├── dnd5e/
└── custom/
```

DND5E 模块内部可拆分：

- ability
- proficiency
- skills
- spellcasting
- rests
- conditions

---

# 5 设计原则

1. 先做计算辅助，不做完整规则裁决。
2. 允许用户手动覆盖计算结果。
3. 复杂规则后续逐步模块化。
4. 不将所有规则写死在角色表中。
5. 保持规则集扩展能力。
