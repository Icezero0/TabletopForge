# 资源库模块设计

版本：v0.2  
状态：Draft

---

# 1 模块目标

资源库模块用于管理用户和房间中的图片资源，为角色头像、地图背景和 Token 提供素材来源。

---

# 2 资源类型

- 角色头像
- Token 图片
- 地图图片
- NPC 图片
- 怪物图片
- 道具图片
- 标记图片
- 状态图标

---

# 3 资源范围

## 3.1 个人资源

仅属于用户本人，可在不同房间中复用。

## 3.2 房间资源

属于某个房间，可供房间成员按权限使用。

---

# 4 功能范围

- 上传资源
- 查看资源
- 重命名资源
- 删除资源
- 资源分类
- 从资源创建 Token

---

# 5 权限规则

- 用户可管理自己的个人资源。
- DM 可管理房间资源。
- 玩家是否可上传房间资源由房间设置决定。

---
