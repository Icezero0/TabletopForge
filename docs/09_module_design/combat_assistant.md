# 战斗辅助模块设计

版本：v0.1  
状态：Draft

---

# 1 模块目标

战斗辅助模块用于辅助 DND 战斗流程，管理先攻、回合、轮数和战斗状态。

本模块不是第一阶段核心，但需要在设计上预留扩展空间。

---

# 2 后续功能

- 开始战斗
- 结束战斗
- 先攻列表
- 当前回合
- 当前轮数
- 下一回合
- 怪物状态追踪
- 状态持续时间
- 专注检定提醒
- 死亡豁免
- 短休 / 长休恢复

---

# 3 关联模块

战斗辅助模块依赖：

- 角色卡模块
- 角色状态模块
- Token 模块
- 骰子模块
- 操作日志模块

---

# 4 权限规则

- DM 可管理战斗流程。
- 玩家可查看战斗流程。
- 玩家可对自己的角色执行允许的状态操作。

---

# 5 后续设计重点

- Initiative 数据模型
- Combat Encounter 数据模型
- Turn 状态同步
- 状态持续时间自动更新
- 战斗日志展示
