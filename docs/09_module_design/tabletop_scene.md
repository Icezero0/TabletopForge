# 场景与地图模块设计

版本：v0.2  
状态：Draft

---

# 1 模块目标

场景与地图模块用于维护房间内不同游戏场景、背景地图和网格配置。

---

# 2 核心概念

## 2.1 Scene

一个房间中的游戏场景，例如酒馆、森林、地下城房间。

## 2.2 Map

场景的背景地图与网格配置。

## 2.3 Current Scene

房间当前激活的场景。

---

# 3 功能范围

第一阶段支持：

- 创建场景
- 编辑场景名称
- 上传地图
- 设置地图尺寸
- 设置网格大小
- 开关网格
- 切换当前场景

后续支持：

- 场景排序
- 场景归档
- 地图测距
- 战争迷雾
- 动态光照

---

# 4 权限规则

- DM 可创建、编辑、切换场景。
- 玩家默认只读场景。
- 旁观者默认只读场景。

---

# 5 实时事件

- SCENE_CREATED
- SCENE_UPDATED
- SCENE_CHANGED
- MAP_UPDATED

---

# 6 设计原则

1. 房间可以有多个场景。
2. 同一时间只有一个当前场景。
3. 地图配置和 Token 数据分离。
4. 初始版本只支持方格网格。
